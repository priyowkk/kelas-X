Belajar HTML dan CSS

HTML digunakan untuk membuat website. CSS digunakan untuk mengatur tampilan

Materi HTML

  • h1 untuk judul
  • p untuk paragraf
  • img untuk gambar
  • table untuk tabel
  • input untuk form
  • li untuk list

Materi CSS

  1. margin,padding,position untuk mengatur jarak antar elemen
  2. display untuk mengatur eleman html agar tampil atau tidak tampil, mendatar (kolom),menurun (baris)
  3. style untuk mengatur font
